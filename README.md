# Установка
* Скачиваем архив
* Распаковываем в удобную нам директорию
* Запускаем CMD
* Переходим в корень с бомбером
* Пишим pip install requirements.txt

# Запуск
* python HZF Bomber.py


# Важная информация
* Бомбер работает ролько на Windows и Linux
